	#Distinct-Reversed Strings

Counts the number of strings that are both distinct and distinct-reversed
within the array, that is, a string is not counted as another distinct string 
if it matches any other string or the reverse of any other string in the array. 

	Examples:

	{"Hi", "iH"}
	1

	{"123", "0", "321", "0"}
	2

	{"hdfjk", "jkfl", "sduho", "fjdkl", "lkjf"}
	4
